Cluedle

Um jogo rápido sobre identificar pistas para encontrar o culpado antes que seja tarde demais.



Gênero: Puzzle
Sinopse: Um jogo investigativo que mistura elementos do jogo Wordle para criar uma situação diária onde o jogador deve, em até seis tentativas, descobrir qual é o personagem procurado como criminoso. A cada dia um novo personagem é escolhido, mas não há limite de repetição, o que mantém a imprevisibilidade do jogo. A cada tentativa errada, o jogador recebe uma dica referente a um dos atributos possuídos pelo personagem do dia, e cabe a ele juntar todas as dicas necessárias para encontrar o suspeito. Cada personagem tem cinco atributos distintos, permitindo que seja possível receber uma dica sobre cada atributo antes de alcançar a última tentativa do jogo. Todo personagem presente é único.
