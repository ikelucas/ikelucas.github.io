Bolsodex

Um projeto que visa traduzir os nomes dos Pokémon para português brasileiro de acordo com sua origem e etimologia.



Link para acesso: Vinizinho (Neocities)

Gênero: Adaptação para o português brasileiro

Sinopse: Utilizando enciclopédias que listam cada Pokémon e explicam a origem de seus nomes, é possível traçar parelelos e chegar em adaptações dos nomes se estes fossem traduzidos para o português brasileiro; o projeto foi feito para criar uma lista com os nomes traduzidos dos bichinhos da primeira e da segunda geração, além de um pequeno texto com uma história fantasiosa que busca simular como teria sido a negociação inicial para aplicar as traduções de maneira oficial. Este projeto já possui mais de 1,5 milhões de visualizações, contando o site original e vídeos no YouTube reagindo ao conteúdo.
