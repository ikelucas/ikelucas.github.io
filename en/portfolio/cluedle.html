Cluedle

A fast-paced game about identifying clues to find the culprit before it's too late.



Genre: Puzzle
Summary: An investigative game that adapt the concept of a Wordle game to create a daily situation where the player must discover which character is the criminal of the day, using up to six attempts. Each day a new character is chosen, but there is no limit on repetition, which mantains the unpredictability of the game. For each wrong attempt, the player receives a hint referring to one of the attributes possessed by the character of the day, and it's up to the player to gather all the necessary hints to find the suspect. Each character has five different attributes, making it possible to receive a hint for every attribute before the game's ending. Every character from the roster of possibilities is unique.
